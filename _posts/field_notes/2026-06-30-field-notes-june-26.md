## What I learned
This month I started digging deeper into Excel. These are the shortcuts and concepts I found myself using enough that I wanted to write them down. 

### Short Cuts
- CTRL + ↓ — Jump to the last populated cell in the current column.
- CTRL + → — Jump to the last populated cell in the current row.
- CTRL + ↑ — Jump to the first populated cell in the current column.
- CTRL + ← — Jump to the first populated cell in the current row.
- CTRL + G (“go to”) - Quickly jump to any cell, named range, or special selection.
- Name box - Located to the left of the formula bar. Jump directly to a cell (for example, D450) without scrolling.
- Shift + F11 - creates new sheet within current work book

### Definitions
BODMAS: (order of operations) Brackets, Orders (i.e., powers and roots), Division, Multiplication, Addition, and Subtraction
